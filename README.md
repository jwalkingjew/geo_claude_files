# Geo Claude Files

Claude instruction files and documentation for Geo Protocol projects.

## Folders

- **`content_management_claude_files/`** - Claude files from the [content-management](https://github.com/geobrowser/content-management) repo. Includes CLAUDE.md, knowledge graph ontology spec, and docs for pagination and scripts.
- **`retrosheet_claude_files/`** - Claude files from the [retrosheet_x_geo](https://github.com/geobrowser/baseball) repo. Includes CLAUDE.md, GRC-20 spec, knowledge graph ontology spec, and SDK/API reference docs.
